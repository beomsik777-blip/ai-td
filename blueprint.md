# Beom Random TD - Project Blueprint

## Overview
Beom Random TD (Hero Defense) is a **premium-tier tower defense game** with high-fidelity 2.5D graphics. It features a sophisticated UI, "cool" character assets, and immersive visual effects, delivering a high-end gaming experience.

## Features
- **Premium 2.5D Visuals:** High-definition character sprites with dynamic height, soft shadows, and magical auras.
- **Cool Enemies:** Distinct monster archetypes (Orc Warriors, Ogre Knights, Mythic Bosses) with unique visual styles.
- **Epic Buttons:** Modern 3D energy-themed buttons with glassmorphism, glowing borders, and tactile feedback.
- **Persistent State:** Automatic saving to `localStorage` for seamless play across sessions.
- **Hero System:** Four elemental heroes with 5 stages of evolution each, featuring enhanced visual effects at higher levels.
